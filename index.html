How It Works

01

Deposit $ZEC

Send your Zcash to our shielded pool. Your transaction is private and encrypted.

bash 
zcash-cli sendtoaddress \\
  "zt1..." \\
  1.0 \\
  --memo "solana-bridge"

02

Zero-Knowledge Proof

We generate a zero-knowledge proof that you own the funds without revealing your identity.

bash 
zcash-bridge generate-proof \\
  --txid "abc123..." \\
  --amount 1.0 \\
  --recipient "solana-address"

03

Bridge to Solana

Your proof is verified and equivalent value is minted on Solana as private SPL tokens.

bash 
zcash-bridge verify-and-mint \\
  --proof-file proof.json \\
  --amount 1.0 \\
  --recipient "solana-address"

  --output-token "solana-asset-mint" \\

// proof

Proof of Concept

Zero-Knowledge Privacy

All transactions use zk-SNARKs to prove validity without revealing sender, receiver, or amount.

$ zcash-bridge show-proof --txid abc123
Proof {
    public_inputs: [amount_hash, recipient_hash],
    private_inputs: [sender_key, amount, memo],
    verification_key: VK
}

Solana Integration

Native SPL token support with program verification. Fast, cheap, and composable.

$ solana program invoke BRIDGE_PROGRAM_ID \\
  --instruction verify_and_mint \\
  --accounts bridge_account,mint,user \\
  --data zk_proof_bytes

Trustless Bridge

No custodial risk. Smart contracts verify proofs on-chain. You maintain full control.

$ zcash-bridge verify \\
  --proof proof.json \\
  --public-inputs inputs.json \\
  --verification-key vk.bin

Proof verified successfully

Protocol API

Available Endpoints

GET api.getStatus()

Get protocol status and pool statistics

POST api.bridgeZecToSolana(amount, recipient)

Bridge ZEC to Solana

GET api.checkTransaction(txid)

Check transaction status

GET api.getPoolIntegrity()

Get pool integrity report

GET api.getRecentTransactions(limit)

Get recent transactions

POST api.sendPayment(amount, recipient, memo)

Send SOL payment. Returns transaction signature on success.

GET oracle.getPriceFeed(symbol)

Get real-time price feed for a cryptocurrency symbol (BTC, ETH, SOL, etc.)

GET oracle.getAllPriceFeeds()

Get all available price feeds with aggregated data from multiple sources

POST oracle.submitDataFeed(feedId, data, nodeAddress)

Submit custom data feed to the oracle network

POST oracle.registerNode(nodeAddress, metadata)

Register a new oracle node in the network

POST oracle.stake(nodeAddress, amount)

Stake SOL to become a qualified oracle node operator

GET oracle.getStats()

Get oracle network statistics (nodes, staked amount, feeds, etc.)

POST oracle.verifyFeedProof(feedId, entryIndex)

Verify cryptographic proof for a feed entry. Returns verification status and proof details.

GET oracle.getProofStatus(feedId)

Get proof verification status for a feed including coverage and consensus.
